  • getRandomInt(min: number, max: number): number
    • Defined in tsutils.ts:38

    Returns a random integer between min (inclusive) and max (inclusive). The value is no lower than min (or the next integer greater than min if min isn't an integer) and no greater than max (or the next integer lower than max if max isn't an integer). Using Math.round() will give you a non-uniform distribution!
